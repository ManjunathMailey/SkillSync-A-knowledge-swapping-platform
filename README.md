## **Getting Started**  
### **Prerequisites**  
Ensure you have the following installed:  
- Node.js  
- MongoDB  
- Git  

### **Installation**  
1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/SkillSync.git
   cd SkillSync
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Start the backend server:  
   ```bash
   cd backend
   npm start
   ```
4. Start the frontend:  
   ```bash
   cd frontend
   npm start
   ```
